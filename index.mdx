## Setting up

**General Steps for Using the OpenSearch API**

1. **Access OpenSearch:** Connect to your OpenSearch cluster's endpoint.
2. **Authentication:** For secure clusters (default in production environments), you need to provide credentials (username/password or an API key) with your requests.
3. **Send HTTP Requests:** Use standard HTTP methods (`GET`, `POST`, `PUT`, `DELETE`) to specific API endpoints.
